Checksum Algorithms

MD5

The MD5 message-digest algorithm is a widely used cryptographic hash function producing a 128 bit hash value, typically expressed in text format as a 32 digit hexadecimal number. This is the most commonly used checksum algorithm and is ideal for most files.

Terminal command to create MD5 checksum:

md5 -r /Users/Username/Desktop/image.tif

Output:

4ae9ea162e3bc88bb4de06996f9a8148 /Users/Username/Desktop/image.tif

SHA-256

With an output hash value of 256 bits (64 digit hexadecimal number), SHA-256 requires more relative processing time than MD5 and is ideally suited for larger files and those with a more complex structure like video.

Terminal command to create SHA-256 checksum:

shasum -a 256 /Users/Username/Desktop/video.mkv

Output:

ebecf1ed49baf94555e8a6313df254dfe57eb4d2209820392806aa0bd31db0a9  /Users/Username/Desktop/video.mkv
